Acta de Defunción

¿Qué es? El Acta de Defunción es el documento oficial que certifica el fallecimiento de una persona, necesario para trámites legales posteriores.

¿Cómo obtenerlo? Se debe presentar el certificado médico de defunción y acudir al Registro Civil para registrar el fallecimiento y obtener el acta.

Costo: Generalmente el trámite tiene un costo aproximado de $250 MXN, aunque puede variar según la localidad.

¿Dónde obtenerlo? En el Registro Civil 02 ubicado en Colonia Santa Elena, Barrio Hojalateros.

Importancia: Este documento es fundamental para realizar trámites de seguro, pensiones, herencias y otros procedimientos legales relacionados.
